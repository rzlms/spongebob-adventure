## 🎮 About the Game
SpongeBob Adventure is a fun platformer game where you control SpongeBob SquarePants!

### 🎯 Features
- 🧽 Play as SpongeBob with square head and sponge holes
- 🍔 Eat 30 Krabby Patties to grow and EXPLODE!
- 👾 Fight enemies: Patrick, Mr. Krabs, Octopus, Jellyfish
- 🌊 10 different underwater environments
- 💥 Explosion effects with particles
- 🎵 Sound effects (eating, exploding, laughing)

### 🕹️ Controls
| Key | Action |
|-----|--------|
| `←` `→` | Move Left/Right |
| `↑` `↓` | Move Up/Down |
| `SPACE` | RAGE Mode! |
| `ESC` | Quit Game |

- Python 3.12+
